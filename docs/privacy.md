# Privacy Policy

This website is the landing page for the Randolph Glacier Inventory (RGI), hosted on [GitHub Pages](https://pages.github.com/) at `glims-rgi.github.io`.

## Data collected

This site does not use user accounts, contact forms, or cookies.

We use privacy-friendly Plausible Analytics to measure aggregate website traffic.Plausible does not use cookies and does not collect personally identifiable information.

## GitHub Pages

This site is hosted by GitHub Pages. GitHub may collect technical information (such as IP addresses) as part of its standard hosting infrastructure. Please refer to [GitHub's Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement) for details.

## External links

This site contains links to external services including [NSIDC](https://nsidc.org), [GLIMS](https://www.glims.org), and [IACS](https://cryosphericsciences.org/). These services have their own privacy policies, which we encourage you to review.

## Contact

If you have questions about this privacy policy, please email [contact@rgidata.org](mailto:contact@rgidata.org).
